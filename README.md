# AI Chatbot System

## Overview

This project is an AI-powered Chatbot System developed using Flask, Machine Learning, NLP, and MySQL.

The chatbot analyzes user questions and predicts the most relevant answer using Natural Language Processing techniques. It also provides emotional support responses for users expressing stress, worry, or sadness.

The project was successfully developed through effective teamwork, collaboration, and coordinated development efforts.

# Machine Learning Concepts Used

This project uses:

- TF-IDF Vectorization
- Cosine Similarity
- NLP-based Text Matching

The chatbot compares user questions with stored dataset questions and predicts the best matching response.
